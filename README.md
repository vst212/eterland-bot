# 使用方法:
  登陆游戏
  在Chrome浏览器中打开: (Mac上) Cmd + Option + J或者(Windows上) Ctrl + Shift + J或F12
  复制代码并按回车键执行即可

  判断是否有土丘，脚本協助清除
